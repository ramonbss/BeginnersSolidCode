# TicTacToe

## 1 - TicTacToe Game

    1.  Select game type

        - Player vs Player
        - Player vs Computer
        - Computer vs Computer

    2. Start game
        1. Show actual board
        2. Next player do it move
            1. Show movement instructions
            2. Ask player for it movement
            3. Check if movement is valid
            4. If not, go to step 2.1
        6. Check if have a winner
            - If not, go to step 1

## 2 - Board

    - Build new board
    - Print board
    - Check board for a winner
        1. Check if game already finished
        2. Check rows
        3. Check columns
        4. Check diagonals

## 3 - Player interface

    - Set player symbol
    - Pick a board position
